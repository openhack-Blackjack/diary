## 사이트 소개

 본 사이트는 다이어리 사이트로 사용자가 다이어리를 작성하면 부가적인 정보들을 같이 저장하여
나중에 다시 일기장을 열어볼때 저장되어 있던 부가 정보들을 보여줌으로서 사용자에게 새로운 감성을 느끼게 해주는 웹 사이트이다.
부가적인 정보로는 날씨, 뉴스기사, 등등을 포함하며 카카오톡 대화 내용을 분석해 가장 대화를 많이한 친구, 그날 공유한 URL을 보여주는것이  특징이다
